## Key background

**Test helper** (already in file):
```javascript
const createTestCPU = () => {
  const mem = new Uint8Array(65536);
  mem[0xFFFE] = 0x10; mem[0xFFFF] = 0x00; // reset vector → 0x1000
  const cpu = CPU6309({
    byteTo: (addr, val) => { mem[addr] = val & 0xFF; },
    byteAt: (addr) => mem[addr] || 0,
  });
  return { cpu, mem };
};
```

**CPU starts at PC=0x1000** after reset. Place opcodes at `mem[0x1000]`.

**Native mode:** Set `mem[0x1000] = 0x11; mem[0x1001] = 0x3D; cpu.singleStep();` to run `LDMD #1` which sets MD bit 0 (native mode). Then advance PC and place next instruction.

**Disassembler call:** `cpu.disasm(b0, b1, b2, b3, b4)` returns `[mnemonic, byteLength]`.

**Flag constants** (from source — these are the CC bit values):
- F_CARRY = 0x01, F_OVERFLOW = 0x02, F_ZERO = 0x04, F_NEGATIVE = 0x08
- F_IRQMASK = 0x10, F_HALFCARRY = 0x20, F_FIRQMASK = 0x40, F_ENTIRE = 0x80
