> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

- [ ] **Step 5: Update TFREXG size-check logic**

The existing TFREXG checks `bit 3` of each nibble for 8-bit vs 16-bit. The new encodings:
- 8-bit: A(8), B(9), CC(A), DP(B), E(E), F(F), zero-8bit(C/D in 8-bit context)
- 16-bit: D(0), X(1), Y(2), U(3), S(4), PC(5), W(6), V(7), zero-16bit(C/D in 16-bit context)

The `is8Bit` check: bit 3 of the nibble = 1 means 8-bit for values 8-F. W(6) and V(7) are 16-bit (bit 3 = 0). E(E) and F(F) have bit 3 = 1 → correctly identified as 8-bit. Zero register (C/D = 1100/1101) has bit 3 = 1 → treated as 8-bit in 8-bit context. The existing check `(ucPostByte & 0x88)` logic should still work for the new registers since E/F (bit3=1) match A/B pattern, and W/V (bit3=0) match D/X pattern.

Verify the `TFREXG` function still correctly rejects mixed 8-bit/16-bit. No change needed if the bit-3 check still holds.

        // LDBT: dst=bit5 of B, src=bit3 of mem[$50], reg=B(01)
        // Postbyte: bits 7-5 = dstBit, bits 4-2 = srcBit, bits 1-0 = reg
        // dstBit=5 -> bits 7-5 = 0b10100000 = 0xA0
        // srcBit=3 -> bits 4-2 = 0b00001100 = 0x0C
        // reg=B=01 -> 0x01
        // postbyte = 0xA0 | 0x0C | 0x01 = 0xAD
        mem[0x1000] = 0x11;
        mem[0x1001] = 0x36; // LDBT
        mem[0x1002] = 0xAD;
        mem[0x1003] = 0x50;

        // B bit 5 = mem bit 3 = 1
        assert.ok(cpu.status().b & 0x20, "B bit 5 loaded from mem bit 3");

            case 0x30: case 0x31: case 0x32: case 0x33:
            case 0x34: case 0x35: case 0x36: case 0x37: {
              // Bit operations (DIRECT mode, special postbyte)
              const pb = fetch();
              const addr = (DP << 8) | fetch();
              const dstBit = (pb >> 5) & 0x7;
              const srcBit = (pb >> 2) & 0x7;
              const regCode = pb & 0x3;

              const getR = () => regCode === 0 ? rA : regCode === 1 ? rB : CC;
              const setR = (v) => {
                if (regCode === 0) rA = v & 0xFF;
                else if (regCode === 1) rB = v & 0xFF;
                else CC = v & 0xFF;
              };

              const memBit = (byteAt(addr) >> srcBit) & 1;
              const regVal = getR();
              const regBit = (regVal >> dstBit) & 1;
              let newBit;
              const op = opcode & 0x7;
              if (op === 0) newBit = regBit & memBit;          // BAND
              else if (op === 1) newBit = regBit & (~memBit & 1); // BIAND
              else if (op === 2) newBit = regBit | memBit;     // BOR
              else if (op === 3) newBit = regBit | (~memBit & 1); // BIOR
              else if (op === 4) newBit = regBit ^ memBit;     // BEOR
              else if (op === 5) newBit = regBit ^ (~memBit & 1); // BIEOR
              else if (op === 6) newBit = memBit;              // LDBT
              else {                                            // STBT (op===7)
                const memVal = byteAt(addr);
                byteTo(addr, (memVal & ~(1 << dstBit)) | (regBit << dstBit));
                break;
              }

              if (op !== 7) {
                setR((regVal & ~(1 << dstBit)) | (newBit << dstBit));
              }
              break;
            }
